PLANK HELPER
======================
### 개요
집에서 플랭크를 할 때 스마트폰의 제조사 기본 스탑워치 어플을 통해 시간을 잽니다.<br>
하다보니 1분 넘게 하는데, 마냥 시계만 보면서 하는 게 지루해 인터넷도 보고 웹툰도 보게 됐습니다.<br>
하지만 다른 작업 중 시간이 얼마나 지났는지 편하게 알 수 없어 불편해 알림창을 통해 제어하고자 개발하게 됐습니다.<br>
